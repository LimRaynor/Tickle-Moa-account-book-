# Views Package Guide

## 역할
- 실제 화면 컴포넌트

## 주요 파일
- `Main.vue`: 메인 화면
- `AboutView.vue`: 소개 화면
- `Signup.vue`: 회원가입
- `LoginView.vue`: 로그인
- `AccountView.vue`: 계좌 관리
- `TransactionView.vue`: 거래 관리

## 사용자 시나리오 흐름
1. `Signup.vue`에서 회원가입
2. `LoginView.vue`에서 로그인
3. `AccountView.vue`에서 계좌 생성/조회/삭제
4. `TransactionView.vue`에서 계좌 선택 후 거래 생성/조회/삭제
